# Crypto Trading Bot

A cryptocurrency trading bot with automated strategies.

## Features

- Real-time market data monitoring
- Automated trading strategies
- Risk management
- Portfolio tracking

## Setup

1. Install dependencies:
```bash
npm install
```

2. Configure your API keys in `.env`

3. Run the bot:
```bash
npm start
```

## Configuration

Copy `.env.example` to `.env` and fill in your exchange API credentials.
